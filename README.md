# Etch - Browser Extension

Bookmark any moment in a ChatGPT or Claude conversation and jump back instantly. Also works on Gemini, Grok, Copilot, Perplexity, Mistral, and more.

## Install

- [Chrome Web Store](https://chromewebstore.google.com/detail/enncblmoicoleaocogkpkabofpdlojho)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/developers/addon/etch/)

## Current Version

**1.23.4**

[Privacy Policy](./PRIVACY_POLICY.md)

## Known Limitations

- **Iframes**: Bookmarks cannot be placed inside iframe-embedded content (for example spreadsheet tables rendered inside a chat). This is a browser security restriction.

## Reporting a Bug

Use the issue template below. Please include:

1. What you were doing when the bug occurred
2. Exact steps to reproduce it
3. What you expected vs. what actually happened
4. Your browser name + version, and the Etch extension version (visible in the options page or the extensions list)
